Deployment-Vorbereitung für den aktuellen Branch:
1. `npm run lint` ausführen — alle Fehler müssen behoben sein
2. `npm run typecheck` ausführen — keine Type-Errors erlaubt
3. `npm run test` ausführen — alle Tests müssen bestehen
4. `npm run build` ausführen — Build muss erfolgreich sein
5. Wenn alle Checks bestehen, erstelle einen PR mit beschreibendem Titel
   basierend auf den Änderungen. PR-Beschreibung auf Deutsch.
