# Context Engineering

## 1. Goal
To mitigate context degradation ("Context Rot") by utilizing hierarchical digest techniques.

## 2. Implementation
- **L0 (Abstract/Topology)**: A high-level view of the directory structure to identify relevant areas quickly.
- **L1 (Structural/Signatures)**: Function and class signatures for deep API understanding without implementation details.
- **L2 (Full Content)**: Exact file contents sent strictly via Dynamic Mutual Context Tailoring (DMCT) only when completely necessary.

## 3. Orchestration
Jules uses these techniques to formulate precise queries, while Claude Code executes tasks with minimalistic, targeted information.
